# 2026-BankCardScan-YOLO-PaddleOCR
随着移动支付的普及，传统人工录入银行卡号的方式效率低且易出错，而早期图像处理方法难以应对复杂卡面与光照变化等问题。为此，本文设计并实现了一套基于深度学习的银行卡号自动识别系统。系统采用 YOLOv26 模型实现卡号区域的高精度快速定位，并引入 CLAHE 对比度增强与高斯滤波等预处理技术，有效克服了反光与低光照干扰。随后，利用轻量级 PaddleOCR 模型进行端到端字符提取，并结合正则表达式规范化输出。同时，本文基于 PySide6 开发了支持图片、视频及摄像头实时流输入的多模态交互界面，提供核心参数联动调节等功能。测试表明，系统在主流 GPU 环境下，单张图片整体处理耗时控制在 57 毫秒以内，识别准确率达 94.3%，有效兼顾了实时性与准确性，具有较高的工程应用价值。
